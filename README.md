# UHaul Load Planner

Interactive overhead floor plan for planning a 26' U-Haul move.

## Features

- Drag-and-drop items on a to-scale floor plan (1 ft = 34 px)
- Add, edit, and delete items with custom dimensions, colors, and shapes
- Feet/inches input support for dimensions
- Snap-to-grid and item rotation
- Real-time floor usage stats (sq ft used / remaining)
- Persistent layout saved via Supabase across visits

## Truck Specs

- Main floor: 23'6" × 8'1" (~190 sq ft)
- Mom's Attic: 2'7" × 8'1" (over cab)
- Door opening: 7'9" W × 6'10" H
- Max load: 9,010 lbs

## Tech

Vanilla HTML/CSS/JavaScript · Supabase
